# Scripts Reference

The `scripts/` directory contains one-off utilities for setup and maintenance.

## Available Scripts

### Account Hash Generation (`scwhab_hash.py`)

**Purpose**: Generate and store the Schwab account hash required for API calls.

**Usage**:
```bash
python scripts/scwhab_hash.py
```

**When to run**:
- Initial setup
- After account changes
- If hash becomes invalid
